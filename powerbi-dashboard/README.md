# Power BI Marketing Dashboard

## Project Overview

This Power BI dashboard analyzes the distribution of the marketing budget and the effectiveness of promotion channels across different cities.

The report helps evaluate how marketing investments are allocated and how they influence the number of attracted buyers.

## Dashboard Features

The dashboard includes several visualizations:

- **Budget dynamics** – shows how the marketing budget changes across quarters
- **Budget by city and promotion channel** – compares advertising investments across cities and channels
- **Number of attracted buyers by promotion channel** – analyzes the effectiveness of different promotion channels
- **Geographical distribution of buyers** – map visualization showing the number of attracted buyers by city

## Tools Used

- Marketing data analysis
