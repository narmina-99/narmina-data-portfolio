# SQL Analysis (PostgreSQL)

## Project Overview

This section of the project demonstrates SQL skills using PostgreSQL.  
The scripts include examples of data selection, filtering, table creation, joins, aggregations, subqueries, and window functions.

The goal of this work is to practice querying relational databases and performing analytical tasks using SQL.

## Topics Covered

- Data selection and filtering (SELECT, WHERE, LIKE, ILIKE)
- Working with database objects (CREATE TABLE, ALTER TABLE)
- Data manipulation (INSERT, UPDATE)
- Table joins and aggregations
- Subqueries
- Window functions

## Files

01_select_where_like.sql – basic data selection and filtering queries  
02_ddl_dml_basics.sql – table creation and data manipulation operations  
03_joins_groupby.sql – joins, aggregations and analytical queries  
04_subqueries_window_functions.sql – advanced queries using subqueries and window functions
