# Python Data Cleaning & Visualization

## Project Overview

This project demonstrates data cleaning and exploratory data analysis using Python.  
The notebook processes marketing-related datasets and combines multiple data sources for analysis and visualization.

## Steps Performed

- Data loading from CSV files
- Data merging using pandas
- Initial data inspection
- Data cleaning and preprocessing
- Exploratory data analysis
- Visualization of key patterns

## Tools & Libraries

- Python
- pandas
- matplotlib
- seaborn

## Files

analysis.ipynb – Jupyter notebook containing the full analysis  

Note: Source CSV files are not included in the repository.
